Description

This project contains solutions to recursion problems in Python.
All tasks are solved using recursive functions only (without loops).

Recursion is a method where a function calls itself until a base condition stops the process.

---

Task 1 — Print digits of a number

The function prints each digit of a number on a separate line using recursion.

Example:
Input: 5481
Output:
5
4
8
1

---

Task 2 — Average of elements

The recursive function calculates the sum of array elements.
Then the average value is calculated using the sum.

Example:
Input: 3 2 4 1
Output: 2.5

---

Task 3 — Prime number check

The function checks whether a number is prime.

A prime number is divisible only by 1 and itself.

Example:
Input: 7
Output: Prime

---

Task 4 — Factorial

The recursive function calculates factorial:

n! = n × (n−1) × (n−2) × ... × 1

Example:
Input: 5
Output: 120

---

Task 5 — Fibonacci number

The recursive function finds the n-th Fibonacci number.

Formula:

F(n) = F(n−1) + F(n−2)

Example:
Input: 5
Output: 5

---

Task 6 — Power function

The recursive function calculates aⁿ.

Example:
Input: 2 10
Output: 1024

---

Task 7 — Reverse output

The recursive function prints array elements in reverse order.

Example:
Input: 1 4 6 2
Output: 2 6 4 1

---

Task 8 — Check digits in string

The function checks whether a string contains only digits.

Example:
Input: 123456
Output: Yes

Input: 12a34
Output: No

---

Task 9 — Count characters in string

The recursive function counts the number of characters in a string.

Example:
Input: recursion
Output: 9

---

Task 10 — Greatest Common Divisor (GCD)

The recursive function finds the GCD using the Euclidean algorithm.

Formula:

gcd(a, b) = gcd(b, a % b)

Example:
Input: 32 48
Output: 16
